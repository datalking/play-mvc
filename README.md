# play-servlet-rest   
>基于servlet的简单易用的rest服务框架   

## target
- 简单易用
- rest服务

## note
- 目前仅支持json接口  

## dev 
```sh
 git clone https://github.com/datalking/play-servlet-rest.git
 cd play-servlet-rest/
 ./start-build-dev.sh
```
